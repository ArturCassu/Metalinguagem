keyup

O evento keyup no JavaScript é disparado quando uma tecla do teclado é liberada após ser pressionada. Esse evento é muito útil em interações de usuário onde é necessário detectar quando o usuário termina de digitar uma tecla, como em formulários de busca ou validação de entrada de dados em tempo real.

exemplo:

input.addEventListener('keyup', verificarInput); nesse caso, o evento keyup será disparado quando o usuário pressionar uma tecla. E isso ira ativar um método que eu criei para verificar se o que o usúario colocou foi digitado corretamente.
